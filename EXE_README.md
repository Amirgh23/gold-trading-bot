# Gold Trading Bot - Standalone EXE

## Quick Start

### Running the Application

Simply double-click `dist/GoldTradingBot.exe` to launch the application.

**File Location**: `dist/GoldTradingBot.exe`
**File Size**: 37.75 MB
**Platform**: Windows 64-bit

## Features

The standalone EXE includes:

- 📊 **Dashboard** - Real-time trading monitoring
- 📈 **Backtesting** - Test strategies on historical data
- 🧪 **Tests** - Run unit and property-based tests
- ⚙️ **Settings** - Configure trading parameters
- 📚 **Documentation** - View guides and documentation
- 🔧 **Tools** - Utility tools (Training, Export, Backup)

## System Requirements

- **OS**: Windows 10 or later (64-bit)
- **RAM**: 4GB minimum (8GB recommended)
- **Disk Space**: 100MB free space
- **Internet**: Required for market data

## First Run

1. Double-click `GoldTradingBot.exe`
2. The application will launch with the GUI Launcher
3. Navigate using the left sidebar menu
4. Configure your trading parameters in Settings
5. Run backtests to validate strategies
6. Monitor live trading in Dashboard

## Configuration

Configuration files are stored in:
- `config/language.json` - Language settings
- `config.json` - Trading parameters (create from `config.example.json`)

## Troubleshooting

### Application won't start
- Ensure you have Windows 10 or later
- Check that you have at least 4GB RAM available
- Try running as Administrator

### GUI not displaying
- Update your graphics drivers
- Ensure PyQt5 libraries are compatible with your system

### Performance issues
- Close other applications to free up RAM
- Reduce the number of indicators being calculated
- Increase the update interval in Settings
