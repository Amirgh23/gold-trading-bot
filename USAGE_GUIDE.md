# Gold Trading Bot - راهنمای استفاده

## 🚀 شروع سریع

### روش 1: استفاده از فایل EXE (ساده‌ترین)

```
1. فایل dist/GoldTradingBot.exe رو دوبار کلیک کنید
2. برنامه بلافاصله شروع میشود
3. منتظر بمانید تا GUI بارگذاری شود (اولین بار ممکن است 10-15 ثانیه طول بکشد)
```

### روش 2: اجرا از Python (اگر Python نصب دارید)

```bash
# 1. Repository رو clone کنید
git clone https://github.com/Amirgh23/gold-trading-bot.git
cd gold-trading-bot

# 2. Dependencies رو نصب کنید
pip install -r requirements.txt

# 3. برنامه رو اجرا کنید
python run_launcher.py
```

## 📊 بخش‌های برنامه

### 1. Dashboard (داشبورد)
**برای**: نمایش real-time trading و وضعیت بازار

**ویژگی‌ها:**
- نمایش قیمت فعلی طلا
- نمایش سیگنال‌های تریدینگ
- نمایش موقعیت‌های باز
- نمایش سود/زیان
- نمایش تاریخچه معاملات

**نحوه استفاده:**
1. از منوی سمت چپ "Dashboard" رو انتخاب کنید
2. اطلاعات real-time نمایش داده میشود
3. برای تازه‌سازی، صفحه رو refresh کنید

---

### 2. Backtesting (تست عملکرد)
**برای**: تست استراتژی‌ها روی داده‌های تاریخی

**پارامترهای تنظیم:**
- **Start Date**: تاریخ شروع تست
- **End Date**: تاریخ پایان تست
- **Initial Capital**: سرمایه اولیه (مثلاً 10,000 دلار)

**نحوه استفاده:**
1. "Backtesting" رو انتخاب کنید
2. تاریخ‌ها و سرمایه رو تنظیم کنید
3. دکمه "Run Backtest" رو کلیک کنید
4. نتایج نمایش داده میشود:
   - Total Return: درصد سود کل
   - Sharpe Ratio: نسبت ریسک/پاداش
   - Max Drawdown: بیشترین کاهش
   - Win Rate: درصد معاملات سودآور

**مثال نتایج:**
```
Total Return: 25.5%
Sharpe Ratio: 1.8
Max Drawdown: 12.3%
Win Rate: 62%
```

---

### 3. Tests (تست‌ها)
**برای**: اطمینان از صحت کد و الگوریتم‌ها

**انواع تست‌ها:**
- **Unit Tests**: تست توابع جداگانه
- **Property-Based Tests**: تست خصوصیات عمومی
- **Risk Manager Tests**: تست مدیریت ریسک
- **Ensemble Router Tests**: تست مسیریابی استراتژی‌ها
- **Position Manager Tests**: تست مدیریت موقعیت‌ها
- **Indicators Tests**: تست شاخص‌های تکنیکال

**نحوه استفاده:**
1. "Tests" رو انتخاب کنید
2. نوع تست رو از لیست انتخاب کنید
3. "Run Tests" رو کلیک کنید
4. نتایج نمایش داده میشود

**معنی نتایج:**
- ✓ = تست موفق
- ✗ = تست ناموفق
- Coverage: درصد کد پوشش‌شده

---

### 4. Settings (تنظیمات)
**برای**: تنظیم پارامترهای تریدینگ

**پارامترهای اصلی:**

#### Risk Management (مدیریت ریسک)
```
Max Drawdown (%): 20
- حداکثر کاهش مجاز در سرمایه
- اگر از این حد تجاوز کند، تریدینگ متوقف میشود
```

#### Position Sizing (اندازه موقعیت)
```
Risk per Trade (%): 2
- درصد سرمایه‌ای که در هر معامله ریسک میکنید
- مثال: اگر سرمایه 10,000 دلار و 2% ریسک = 200 دلار ریسک
```

#### Strategy Parameters (پارامترهای استراتژی)
```
Confirmation Threshold: 0.6
- حداقل امتیاز برای تایید سیگنال
```

**نحوه استفاده:**
1. "Settings" رو انتخاب کنید
2. پارامترها رو تنظیم کنید
3. "Save Settings" رو کلیک کنید

### 6. Tools (ابزارها)
**برای**: کارهای کمکی

**ابزارهای موجود:**

برای آموزش مدل LSTM
- داده‌های تاریخی رو استفاده میکند

#### Export Trade History
```
برای صادر کردن تاریخچه معاملات
- فرمت: CSV
- شامل: تاریخ، قیمت، سود/زیان
```

#### Database Backup
```
برای پشتیبان‌گیری از داده‌ها
- تمام داده‌ها رو ذخیره میکند
- در صورت مشکل، میتونید بازگردانید
```

#### Clear Cache
```
برای پاک کردن حافظه موقت
- سرعت برنامه رو بهتر میکند
- داده‌های مهم حذف نمیشود
```

#### Generate Report
```
برای تولید گزارش عملکرد
- شامل: سود، ریسک، استراتژی‌ها
- فرمت: PDF یا Excel
```

## ⚙️ تنظیمات پیشرفته

### فایل Configuration

**مکان**: `config.json`

**مثال:**
```json
{
  "trading": {
    "symbol": "XAUUSD",
    "timeframe": "5m",
    "max_drawdown": 20,
    "risk_per_trade": 2
  },
  "exchange": {
    "name": "MetaTrader5",
    "api_key": "your_key_here"
  },
  "ml": {
    "model_type": "LSTM",
    "retrain_interval": 24
  }
}
```

### فایل Language

**مکان**: `config/language.json`

**تغییر زبان:**
```json
{
  "language": "fa"  // فارسی
  // یا
  "language": "en"  // انگلیسی
}
```

## 🔧 حل مشکلات

### مشکل: برنامه شروع نمیشود

**راه‌حل:**
1. Windows 10 یا جدیدتر دارید؟
2. حداقل 4GB RAM دارید؟
3. به عنوان Administrator اجرا کنید

### مشکل: GUI نمایش داده نمیشود

**راه‌حل:**
1. درایورهای گرافیکی رو آپدیت کنید
2. PyQt5 رو دوباره نصب کنید: `pip install --upgrade PyQt5`

### مشکل: Backtesting خیلی کند است

**راه‌حل:**
1. بازه زمانی کوچک‌تر انتخاب کنید
2. تعداد شاخص‌ها رو کم کنید
3. سرمایه اولیه رو کم کنید

### مشکل: خطا در تریدینگ

**راه‌حل:**
1. API key رو بررسی کنید
2. اتصال اینترنت رو بررسی کنید
3. لاگ‌ها رو بررسی کنید: `logs/trading_bot.json`
